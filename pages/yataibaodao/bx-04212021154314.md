<div id="storytext">
 <p>
  中国维权律师陈科云近日在广州自己家附近被不明身份人员殴打，致全身多处受伤，当地警方已受理案件但尚未积极开展调查。此次陈科云的遭遇并不是维权律师遇袭的第一起，有律师表示，当局没有正当理由通过抓捕、定罪来打压维权律师，因此只能通过下作的手段进行骚扰。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  据维权网消息，4月20日晚九点，中国维权律师陈科云在广州市天河区一住宅小区内散步时，遭遇不明身份人员袭击。陈科云随后前往医院检查，结果显示陈科云身体多处淤血，右侧第8肋骨骨折，左侧第3、5肋骨轻伤，右眼受伤严重。目前广州市公安局已受理报警求助，截至发稿，该案尚无任何进展。
  <br/>
  <br/>
  陈科云在接受本台采访时表示，在此之前当地有关部门曾多次逼迫他搬离现居住地，但均遭到陈科云断然拒绝。陈科云说，此次遇袭恐与这个有关：“第一次是他们在我车上面贴纸，说不能停在这，这个小区我住了五年。第二次是3月25号，他们把我那台车的玻璃砸坏了。”
  <br/>
  <br/>
  陈科云说当地公安局将此案定性为一般自然案件，公安局通知说，陈科云需要提交补充材料才能考虑是否将此案转为刑事案件。他说，目前尚未接到派出所任何案情进展的通知：“我当时就提出来了这很明显是刑事案件，属于寻衅滋事。”
  <br/>
  <br/>
  陈科云作为维权律师长期关注人权法治进步，为弱势群体发声，多次代理人权案件。2013年，陈科云代理杨霆剑煽颠案、刘远东扰乱社会秩序案；2014年为王爱忠等寻衅滋事案辩护等。2017年，陈科云担任轰动一时的桃江四中肺结核传染事件律师团的召集人，因此被注销律师证。
  <br/>
  <br/>
  今年年初，陈科云因前往成都参加维权律师卢思位的吊销执照听证会，被警方关押9小时；不久后，陈科云与维权律师谢阳一同前往宝鸡市探望被抓的维权律师常玮平的父母，被当地国保从高铁站带走软禁超过24小时。
 </p>
 <p>

  <br/>
  流亡美国的维权律师吴绍平告诉本台，维权律师在中国不仅职业发展空间被压缩，其自身权益和人身安全无法得到保障，在中国这已经成为常态：“这是一个很典型的例子，是中共的政权日益流氓化、黑社会化的表征。这种事发生在陈科云律师身上并不是第一起，之前也有（其他）广州律师受到过这样的对待，最后报警都不了了之。这些行为实际上就是中共幕后在指挥，有目的地实施。”
  <br/>
  <br/>
  吴绍平认为，当局没有正当理由和证据通过抓捕、定罪的形式打压多次参与政治敏感事件的陈科云，因此才屡次通过带有黑社会性质的手段在生活中骚扰他。
  <br/>
  <br/>
  吴绍平说：“这种行为不构成寻衅滋事也是故意伤害致人轻伤，这两个罪名无论是哪个，都是涉嫌犯罪的行为。如果是寻衅滋事还致人轻伤，那更是加重情节，也应该立案。中国维权律师的生存现状日益艰难。”
  <br/>
  <br/>
  2009年也曾发生过一起类似的案子。曾代理郭飞雄案的广州维权律师刘世辉就因身穿带有“一党专政、遍地是灾”字样的文化衫，在与同行爬山途中遭遇不明身份人员施暴，随后警方也未进一步调查。
  <br/>
  <br/>
  本台记者一冰 责编：申铧 网编：郭度
