 <p>
  华为在欧洲布局第五代行动通讯技术(5G)之路一再受阻，欧洲联盟第一个明确禁止华为参与5G网路基础设施的国家瑞典，对于华为的上诉，6月22日法院再度宣布维持禁令，使得华为重返欧洲市场的希望破灭。不过瑞典设备制造商爱立信（Ericsson）担忧，此举将引发中国报复，使爱立信流失中国市场。不过，不是只有瑞典坚持这一立场，罗马尼亚在六月初成为最新一个禁止华为参与5G部署的欧洲国家。
  <br/>
  <br/>
  瑞典斯德哥尔摩行政法院22日裁定，维持对中国电信公司华为在瑞典销售5G设备的禁令。
  <br/>
  <br/>
  2020年10月，瑞典电信监管机构邮政及电信管理局(PTS)出人意料地下令禁止华为向瑞典移动公司供应5G设备，主要的原因是瑞典情报安全局(SAPO)对华为提出了安全担忧。瑞典邮电管理局频谱分析设备主任贝克斯(Anne Beckius)就说，“执照条件是在军方及国家安全局的评估后制订的。”
  <br/>
  <br/>
  华为不满这一禁令，随后提起法律诉讼。斯德哥尔摩法院22日裁定，瑞典邮电管理局去年十月决定，禁止在瑞典新的5G电信网路中使用中国公司华为和中兴的设备是合法的。
 </p>
 <p>
  <br/>
 </p>
 <p>
  <br/>
 </p>
 <p>
  行政法院发表声明说，瑞典的安全是一个重要的原因，法官梅林 (Ulrika Melin)引述行政法院的判决指出，国安与国防部门对于瑞典安全的判断有全面的了解。
  <br/>
  <br/>
  华为的北欧执行副总裁费德利克森（Kenneth Fredriksen）表示，法院根据瑞典情报安全局的假定而得出的结论，并不令人意外，华为将继续争取进入瑞典市场的权利。
  <br/>
  <br/>
  北京先前要求瑞典立即纠正禁止华为的错误，并暗示可能对瑞典企业爱立信进行报复。
  <br/>
  <br/>
  营业额约百分之十来自中国市场的瑞典电信商爱立信首席执行官鲍毅康（Bjorn Ekholm）今年1月4日威胁过政府，强调瑞典政府决定排除华为之后，爱立信可能会遭到中国的锁定和报復，担心有可能会因此失去中国市场。
 </p>

  <br/>
  <br/>
  欧洲已成为北京与华府之间的科技冷战战场，但去年七月，欧盟也鼓励成员国停止使用可能增加网络安全风险的供应商，被认为是针对中国公司。而在11日，欧洲又有罗马尼亚禁止中国和华为参与该国5G电信网络的开发。
  <br/>
  <br/>
  罗马尼亚总统伊尔哈尼斯(Klaus Iohannis)11日签署一项法案，将有效阻挡华为参与罗马尼亚的5G网路建设。
  <br/>
  <br/>
  这项法案源自于2019年美国与罗马尼亚的一项备忘录。根据这项备忘录，两国政府表示，基于风险安全对策的一部分，有必要对5G供应商进行慎重和全面的评估，将受到外国政府操控且所有权缺乏透明度的供应商排除在本国的5G网络建设之外。
 </p>
