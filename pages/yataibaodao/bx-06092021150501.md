<div id="storytext">
 <p>
  今年的高考刚刚落下帷幕，各大高校为吸引生源展开激烈争夺战。近日，老牌名校之一的南京大学的官方宣传照在网络上引发巨大争议。身穿露脐装的漂亮女大学生手持标语牌上写着：“你想不想让我成为你的青春？”引发舆论对其物化女性的声讨。
 </p>
 <p>
  本台记者查询，据南京大学的官网显示，这组招生宣传照片是一个系列，包含6张照片。不仅另有女生手持标语显示“你想不想和我一起泡在杜厦图书馆，从早晨到夜晚？”还有男生的照片，手持标语牌上写着“你想不想抚摸南大每一个建筑上历史留下的印记？”“你想不想成为一个‘诚朴雄伟，励学敦行’的南大学子？”等。
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:620px;">
   <img alt="南京大学低俗招生广告。(Public Domain)" src="https://www.rfa.org/mandarin/yataibaodao/kejiaowen/bx-06092021150501.html/68cda150-e492-4ab7-88e7-251df3ce0607.png/@@images/5bd5a026-1aaf-4918-b332-81bf0310b041.png" title="4"/>

    南京大学低俗招生广告。(Public Domain)

  <br/>
  <br/>
  南京大学的招生广告原本想贴近年轻人的生活，却弄巧成拙，其物化女性的嫌疑引来网友一片声讨。有微博网友评论说：“男生举的是梦想和历史印记，是和大学一起成长。女生是‘陪我泡图书馆’、‘让我成为你的青春’，合着女大学生是陪读吗？”还有网友表示不满：“这不就是公然视女性为性资源吗？这个招生广告直到面向大众都没人质疑过吗？” 迫于舆论压力，6月8日，南京大学已将相关招生广告下架。
  <br/>
  <br/>
  苏州前中学教师潘露接受本台采访时表示，高等院校通过具有性暗示的广告来吸引生源是迎合现在低俗的价值观取向：“现在不但像南京大学这样比较好的高校出低俗的招生广告，有些高校在招留学生时也发出带有性暗示的广告，比如山东大学推出学伴政策，来为非洲留学生提供异性学伴。这是整个大陆的文化取向，与高等教育的原则是相违背的。”
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:631px;">
   <img alt="南京大学低俗招生广告。（Public Domain）" src="https://www.rfa.org/mandarin/yataibaodao/kejiaowen/bx-06092021150501.html/22.png/@@images/2f211966-89e6-46d1-8c2f-2b9cb3cc9650.png" title="2"/>
   <figcaption class="image-caption">
    南京大学低俗招生广告。（Public Domain）

   <small>

  <br/>
  <br/>
  自2016年起，山东大学推出学伴制度。在山东大学发布的《关于学历生报名“学伴”项目的通知》中，留学生办公室可以为中国学生介绍一名留学生“学伴”，两人可以共同学习语言或参加活动。2018年，山东大学对学伴制度作出调整，为一名外国留学生配三个中国学伴，其中大多数是外国男留学生配中国女学伴。该事件直至2019年才因发生强迫中国女学生就范的丑闻，在网络上引起轰动。民间舆论认为山东大学物化女学生、格外优待外国来华男留学生。为平息舆论，山东大学国际事务部发布说明作出回应，称学伴制度合法正当，不存在行政强力配对和不良引导，同时山东大学还在说明中暗示南京大学、吉林大学、哈工大等一流院校也开展学伴活动。
  <br/>
  <br/>
  潘露说，中国近些年虽然在物质方面取得长足发展，但伴随而来的是浮躁的社会风气和精神层次的下滑，学生和家长在报考高校时，考虑的因素更加现实：“大部分人看重以后的就业率，在国内人才市场上能获取多高的薪水，这是大部分学生和家长考虑的，他们对于学术研究方面并没有太多的要求。总体而言这是一个物化的趋势，这种趋势在高校中泛滥必然带来价值取向的变化。大陆的高校不但没有外国高校的科研氛围，而且在市场竞争中越来越低俗化，也体现了整个社会的价值取向。”
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:741px;">
   <img alt="南京大学低俗招生广告。（Public Domain）" src="https://www.rfa.org/mandarin/yataibaodao/kejiaowen/bx-06092021150501.html/11.png/@@images/a06e77fe-84d5-4cfd-9539-c21c05039bc8.png" title="2"/>

  <br/>
  用美女照片来吸引新生报考并非首次。早在2013年，中国人民大学一改往日严肃风格，在其官网上发布了女生毕业照，成为最早以美女吸引新生的学校之一。广告一出，人大官网访问量骤升，一度导致网站瘫痪。
  <br/>
  <br/>
  美国南卡大学商学院教授谢田告诉本台，从策划、审批到广告推出，所有环节中竟然没有一个人提出异议，这在国外高校中是不可能发生的事：“在美国不会出现这些，因为他们首先没有用色相或性暗示的东西来吸引学生这种想法，所以也不会做出来。即使做出来也有审批的过程，最终不可能刊登出来，社会舆论也不会接受。美国大学招生广告里一定要强调自己的科研、教学和环境这些真正和学术有关系的东西。（用美女照片吸引生源）这实际上是一种性别歧视。”
  <br/>
  <br/>
  谢田提到，美国教育法修正案第九条规定，任何由联邦资助的教育机构的任何教育计划，包括入学、招生、课程设置等，不能有任何性别歧视行为，一旦发现相关机构违反该法条，联邦经费就会被取消。此外，校内还有相关委员会及组织，积极监察学校行为。
  <br/>
  <br/>
  谢田说：“这就是整个中国社会道德下滑在校园里的反映，实际上中国政府开放性乱，引导社会风气向乱性的方向发展。”
 </p>
 <p>
  最近中国大学校园乱象频现。除了南大的招生广告引发争议之外，复旦大学研究员因被解聘而杀害院党委书记；江浙一些大学附属的独立学院未经与学生商量擅自要与职业大学合并引发学生和家长的抗议示威，并发生警民冲突。
  <br/>
  <br/>
  据高等教育专业评价机构软科发布的2021中国大学排名，南京大学作为老牌高校，今年成功跻身全国第五。在2021QS世界大学排名中，南大位列世界第124位。
 </p>
