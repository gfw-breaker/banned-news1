<div id="storytext">
 <p>
  22日在美国主办的世界领导人气候峰会上，美国总统拜登首次和中国国家主席习近平同台。尽管美国在会前呼吁，中国应在减排上承担更多责任，不过，习近平仍重申中国的碳排放目标没有改变，他还提出“六个坚持”，并称中国已远比许多发达国家做得更多。事实真是如此吗？
  <br/>
  <br/>
  世界地球日的这一天，在美国召集下，身处不同时区的40个国家的领导人，通过预先录制好的录像或是视频连线，参加美国主办的“领导人气候峰会”。重返《巴黎气候协定》的美国，试图在气候问题上展现“美国回来了”的领导力。
 </p>
 <p>
 </p>
 <p>
  <strong>
  </strong>
 </p>
 <p>
  拜登在开幕式上说："在气候变迁这个问题上，不作为的代价日益升高，在领导这个具有道德和经济紧迫性的问题上，美国不会等待。"
  <br/>
  <br/>
  拜登也宣布了美国的雄心勃勃的碳排放目标，也就是到2030年以前，美国将在2005年的基础上减少碳排放50%-52%；2050年前，美国将实现碳中和。
  <br/>
  <br/>
  他意有所指地说，没有哪一个国家可以独自面对气候危机。美国作为碳排放量占世界不到15%的国家，也呼吁所有国家要拿出决心、具体行动，事不宜迟。
  <br/>
  <br/>
  <strong>
   共同但有区别的责任 中国坚称已做很多
  </strong>
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:620px;">
   <img alt="拜登和习近平会见的资料照片。（路透社）" src="https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/rc-04222021150240.html/252d67ca-9cc9-485b-9274-c6b6bfbe4868.jpeg/@@images/9883e3e7-73f7-48c4-a87d-ddbbc1982c66.jpeg" title="1"/>
   <figcaption class="image-caption">
    拜登和习近平会见的资料照片。（路透社）
   </figcaption>
   <small>
   </small>
  </figure>
  <br/>
  <br/>
  中国身为占比达30%的世界碳排放最大国，其国家主席习近平在事先录制好的视频中，则展现如如不动的态势。他提出“六个坚持”，强调在设定碳排放目标上各国应该坚持承担“共同但有区别的责任”的原则，也不忘标榜中国已经做得很多。
  <br/>
  <br/>
  “中国力争2030年前实现碳达峰、2060年前实现碳中和。中国实现从碳达峰到碳中和的时间，远远短于发达国家所用的时间，这需要中方付出艰苦努力。”习近平指出。
 </p>
 <p>
  <strong>
   习近平没说的难听话 葛来仪：美中战略竞争将持续
  </strong>
  <br/>
  <br/>
  值得注意的是，中国官方喉舌新华社随后发出致词全文，却出现习近平呼吁各国要“携手合作，不要相互指责；要持之以恒，不要朝令夕改；要重信守诺，不要言而无信。”这段讲话，习近平在视频中并没有说。
  <br/>
  <br/>
  中国机锋处处，到底想不想和美国合作？美国的德国马歇尔基金会（German Marshall Fund）亚洲项目主任葛来仪（Bonnie Glaser）告诉本台：“中国持续指责是美国一手造成美中关系恶化，还把自己塑造成是全球的治理典范、多边主义与全球化的支持者，即便美中两国在气候变迁问题上有所合作，双方的‘战略竞争’仍会持续。”
  <br/>
  <br/>
  在气候问题上，中国到底做了多少？美国智库威尔逊中心环境计划主任吴岚（Jennifer Turner）长期关注中国环境与美中气候变化合作议题，她接受本台访问时表示：“老实说，许多国际与中国的能源专家都认为，中国已经兑现自身在巴黎气候协定的承诺。当然，中国和美国，身为碳排放世界第一和第二大国，都可以做得更多。”
  <br/>
  <br/>
  <strong>
   绿色一带一路？ 中国己所不欲应勿输出于人
  </strong>
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:620px;">
   <img alt="中国减碳排放的承诺受到质疑（自由亚洲电台制图）" src="https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/rc-04222021150240.html/ac5a9d0f-3fed-4b17-b0be-85455e012b8c.jpeg/@@images/bee57d38-9ce1-42bf-97a4-7da6e28dd7e1.jpeg" title="3"/>
   <figcaption class="image-caption">
    中国减碳排放的承诺受到质疑（自由亚洲电台制图）
   </figcaption>
   <small>
   </small>
  </figure>
  <br/>
  <br/>
  习近平在会中说，全球气候治理应充分肯定发展中国家所做贡献，照顾这些国家的特殊困难与关切；发达国家应展现更大雄心与行动，切实帮助发展中国家加速绿色低碳转型。
  <br/>
  <br/>
  他也强调，中国会坚持走生态优先、绿色低碳的道路，中国支持有条件的地方和重点行业与企业率先达峰，将严控煤电项目；“十四五时期”会严控煤碳消费增长，“十五五时期”将逐步减少。
  <br/>
  <br/>
  吴岚就说，“在‘一带一路倡议’中，过去5到6年之间约有八成的投资项目，中国都是投资在煤炭和石油行业；你可以说发展中国家亟需燃煤发电，但是、但是、但是（因为很重要所以说三次），中国并没有为其他发展中国家带来‘洁净燃煤’发电技术，而是采用了比较便宜、也比较不干净的燃煤发电。”
  <br/>
  <br/>
  吴岚认为，习近平强调了在应对气候问题上要坚持多边主义与体系，这是为美中合作敲开了合作之门，但中国仍会是美国合作型的竞争对手（cooperative competitor）。
  <br/>
  <br/>
  在美中两国可以合作的气候议题上，习近平没说出的新华社稿件的内容，正显示了中国试图且勇于当“世界教师爷”的态度。
  <br/>
  <br/>
  自由亚洲电台记者郑崇生华盛顿报导 责编：申铧 网编：郭度
