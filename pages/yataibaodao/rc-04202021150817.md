<div id="storytext">
 <p>
  美国战略司令部（USSTRATCOM）司令、海军上将理查德20日在国会听证会上说，中国是美国的重大战略威胁，尤其是解放军军力快速现代化、核能力的扩充速度更是前所未见。他坦言，美国不能再假设在冲突中战略威慑失败的风险仍低。事实上，中国近来在南海的行动，正凸显中国正以多样化作为，忽视国际规则与美国的威慑力。
  <br/>
  <br/>
  联邦参议院军事委员会为审理年度预算的听证会上，理查德（Charles Richard）和美军太空司令部（USSPACECOM）司令狄金森（James Dickinson）作证，两人都把矛头指向了中国。
 </p>
 <p>
  理查德在事先准备好的书面报告中明白指出，中国解放军军力现代化的长期目标，是建立区域霸权，阻止美国在印太地区的军力投射，并且取代美国，成为区域国家的安全伙伴。而在中国高度不透明的军事扩张计划下，他更关注中国的核武能力的扩张，正让美国的战略威慑、也就是核威慑能力面临挑战。
  <br/>
  <br/>
  “我别无选择，必须视中国是美国的重大战略威胁。中国已经有能力在亚太地区执行核使用战略（nuclear employment strategy），预计很快也会具备跨洲际的核使用能力。再加上俄罗斯，这是美国史上首次同时面对两个具有核武能力、与美国旗鼓相当的敌手。”理查德说。
  <br/>
  <br/>
  战略司令部是美军联合作战司令部之一，负责太空战、资讯战、防御飞弹系统、情报侦察监视、全球打击、战略威慑、大规模杀伤性武器等领域。
  <br/>
  <br/>
  <strong>
   中国核扩张速度前所未见 美国自缚手脚？
  </strong>
 </p>
 <p>

 </p>
 <p>
  美国身为签署《核不扩散条约》的一员，在美国总统拜登政府上任后，也将美俄之间的《新削减战略武器条约》延长五年。
  <br/>
  <br/>
  理查德坦言，美国核武增加是零的情况下，看着中国的核能力扩张以“前所未见”的速度发展，他亟需国会在例如美国的洲际弹道导弹系统上持续的预算支持，维持美国的现有优势。
  <br/>
  <br/>
  究竟中国军事现代化速度有多快？以核弹头数量为例，理查德无法在公开听证会上给出具体数字，只能闭门向议员简报。但他形容，关于中国军力威胁的情报，不只是每周有报告，且需要每月更新，“因为上个月报告的情资，就已经过时了。”
  <br/>
  <br/>
  美国身处的战略环境确实让理查德不得不忧心，拜登上任以来，中俄分别在印太区域与东欧乌克兰伺机而动，且动作频频。
  <br/>
  <br/>
  <strong>
   中国民兵“小蓝人”扮渔船 南海躲风不肯走
  </strong>
  <br/>
  <br/>
  以中国在南海为例，菲律宾政府上周发布照片，截至4月15日，还有六艘疑似隶属中国民兵的海上“武装渔船”集结在菲律宾专属经济区内的南海牛轭礁（Whitsun Reef）海域。
  <br/>
  <br/>
  长期研究中国民兵部队的美国海军战争学院中国海事研究所教授艾立信（Andrew Erickson）与助理教授马丁森（Ryan D. Martinson）就共同在最新一期《外交政策》（Foreign Policy）撰文指出，这些中国官方声称的渔船，就是隶属于民兵部队的海上武装船只。
  <br/>
  <br/>
  艾立信称中国这种“军民融合”的海上武装力量为“小蓝人”，这是取自俄罗斯入侵乌克兰、以秘密武装人员为主力的小绿人概念。
  <br/>
  <br/>
  文章指出，菲律宾政府发布的照片中有两艘停靠在牛轭礁的“渔船”分别为“粤新会渔60138”和“粤新会渔60139”，显示这些武装船是注册于广东省江门市新会区，名为渔船，但早在2002年，新会渔的相关船只就接受解放军江门军区的海上训练，甚至在2004年成立企业化组织。
  <br/>
  <br/>
  根据《解放军报》，2014年中国在南海上与各邻国争议扩大之际，广东省江门市新会区人武部队更集中组织海上民兵，参加“实战演训”。
  <br/>
  <br/>
  文章认为，这批滞留争议岛礁的船队，就是北京“不愿承认的海军”、更是中国在“灰色地带”的活棋。
  <br/>
  <br/>
  <strong>
   美国警醒 中国巧用军民融合
  </strong>
  <br/>
  <br/>
  在太空与人工智能领域，狄金森就说，应对中国的威胁挑战，尤其在太空与通过卫星系统情报侦搜上，美国确实有必要有一定程度的商用与军用结合。
  <br/>
  <br/>
  狄金森说，“中国绝对对我们目前在太空领域带来步步进逼的挑战，我们要想知道中国究竟在太空领域的意图是什么？他们有什么能力、构成什么样的威胁？我认为，将美国太空商用企业的能力整合进来，是绝对必要的，这是我们美国很大的优势。”
  <br/>
  <br/>
  台湾的《全球防卫杂志》形容，中共的海上民兵部队就是“披着企业外衣的国营海上佣兵”。中国的海上民兵过去也曾出没于钓鱼台海域。
  <br/>
  <br/>
  根据中华人民共和国《民兵工作条例》第五条，全国的民兵工作在国务院、中央军委领导下，由中国人民解放军总参谋部主管，也就是所谓的预备部队。但自2020年7月1日零时起，中国的预备役部队全面纳入军队领导指挥体系，由原本的军地双重领导，调整为“党中央、中央军委集中统一领导”。
  <br/>
  <br/>
  中国国家主席习近平上任后对中国民兵部队力量的重视程度，从他2013年4月就任总书记后立刻造访海南岛海上民兵部队就是一例；同年7月，中国创立南海三沙市海上民兵部队，并在2015年2月于永兴岛注册成立国营三沙市渔业发展公司，但其实就是国家支持的民间军事公司，成为中共干预南海及周边所有权争议海域的非传统手段。
  <br/>
  <br/>
  而这些中国政府支持的海上佣兵还要躲风雨到什么时候？在灰色地带游走的中国，显然没有想在南海问题上展现大国担当，与邻为善的意图。
 </p>
 <p>
  自由亚洲电台记者郑崇生华盛顿报导 责编：申铧 网编：郭度
 </p>
