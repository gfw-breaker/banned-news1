<div id="storytext">
 <p>
  前中国国务院总理温家宝近日发表回忆母亲的长文，内容提到在中南海做官“如履薄冰”，以及“我心目中的中国，应该是一个充满公平正义的国家”等语句，目前该文章已被微信禁止分享，引发关注。
 </p>
 <p>
  中国前总理温家宝近日撰文追忆于去年12月过世的母亲杨志云。长篇文章分四篇于3月25日至4月15日发表于《澳门导报》，文中谈及母亲的一生、对自己的教育、回顾官场打拼以及对中国的期盼。
  <br/>
  <br/>
  温家宝文中写到母亲对他担任总理的心情：“对于我担当大任，妈妈是忧虑的”，除此之外也提及自己在官场打拚的感慨，他称自己在中南海工作 28 年，担任总理10年，“奉命唯谨，如履薄冰、如临深渊，受事之始，即常作归计。”
  <br/>
  <br/>
  而最引人瞩目的是文章结尾，温家宝写道：“我同情穷人、同情弱者，反对欺侮和压迫。我心目中的中国应该是一个充满公平正义的国家，那里永远有对人心、人道和人的本质的尊重，永远有青春、自由、奋斗的气质。我为此呐喊过、奋斗过。”
 </p>
 <p>
  <img alt="3" class="image-richtext image-inline captioned" src="https://www.rfa.org/mandarin/yataibaodao/zhengzhi/cm-04192021151702.html/85775c9e-4990-454e-9dcb-7ccbabcfacda.jpeg" title="3"/>
 </p>
 <p>
  温家宝题为《我的母亲》的这篇文章，至记者截稿时仍可在社交平台微信上看到，却因“该文章违反微信公众平台运营规范”被禁止分享；而根据其他媒体报导，中国媒体凤凰网等原有的转载内容，也全数被删除，引起舆论关注，其中文末关于中国应是“充满公平正义的国家”，更被认为是与现今习近平当局的治理路线越走越远。
  <br/>
  <br/>
  现旅居美国的中共中央党校前教授蔡霞就指出，这篇文章已经做了“自我审查”，长文内没有“民主”及“法治”四个字，即便这样文章依旧被禁止转发分享。
  <br/>
  <br/>
  “可见大陆极权统治当局对‘民主、法治’有多么恐惧。他们恐惧人民的权利。”蔡霞在社交媒体推特上写道。
  <br/>
  <br/>
  2010年，时任中国总理的温家宝至纽约出席联合国大会时，接受了美国有线电视新闻网的专访，谈及许多敏感议题，不过关于“六四”、“人权”以及“互联网封锁”等敏感问题并未在中国媒体上播送。当时温家宝表示，言论自由是“必不可少的”，而中国人民对民主和自由的渴望是“不可避免的”。
  <br/>
  <br/>
  旅美独立学者吴祚来就认为，温家宝的文章可能触动到当局的敏感神经，被视为在抢夺话语权，因此遭到屏蔽。
  <br/>
  <br/>
  “他是总理级的前国家领导人，而且这么温和的文章都被封杀，证明他们（当局）对任何可能暗地里批评，或是有疑似不满的文章，都不允许。”吴祚来告诉本台。
 </p>

  <br/>
  <br/>
  他认为，中共体制难改，但温家宝是真心反文革、求政改，显然与现任政权路线格格不入，因此借由追忆母亲文章“不吐不快”。
  <br/>
  <br/>
  温家宝担任中国总理期间曾多次谈到政治体制改革和普世价值。2012年全国人大闭幕后的记者会是温家宝退休前的最后一场记者会，他当时多次提到“改革”，并表示没有政治体制改革，文化大革命的历史悲剧还有可能重新发生： “不仅要进行经济体制改革，而且要进行政治体制改革，特别是党和国家领导制度的改革。”
  <br/>
  <br/>
  不过，也有许多人将温家宝视为“影帝”、注重作秀，而美国媒体《纽约时报》在2012年的长篇报导中提到，温家宝家族在过去20年中积累了至少27亿美元的资产。中共元老李锐的女儿、定居美国的李南央就批评，温家宝当时的权力比现任总理李克强来得大，却没有任何作为，反倒在退休后继续作秀。
  <br/>
  <br/>
  “他在位子上时就会不一样了，下来以后只有说一说的份，”李南央笑道，“说一说又有什么用呢？他也没说到什么点上，你看他根本就是自我表演。”
  <br/>
  <br/>
