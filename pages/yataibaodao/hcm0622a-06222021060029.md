<div id="storytext">
 <p class="p3">
  美国、台湾、日本与欧盟
  <span class="s2">
   22
  </span>
  日共同举办科技产业全球供应链合作论坛，除了美、日、欧盟驻台代表之外，美国国务院助理次卿莫瑞
  <span class="s2">
   (Matt Murray)
  </span>
  参与论坛提及，“与全球伙伴合作，是美国在加强重供应链韧性方面的重中之重。”
 </p>
 <p class="p3">
  美国在台协会与欧洲经贸办事处、日本交流协会以及台湾经济部在
  <span class="s2">
   22
  </span>
  日共同举办“科技产业全球供应链合作论坛”.
 </p>
 <p class="p3">
  美国国务院助理次卿莫瑞
  <span class="s2">
   (Matt Murray )
  </span>
  在会中提及，面对新冠肺炎疫情供应链的挑战。在今年
  <span class="s2">
   2
  </span>
  月
  <span class="s2">
   24
  </span>
  日美国拜登与贺锦丽执政团队，发布“关键产品供应链行政命令”，希望确保所有美国国民在危机时，能取得关键物资与服务。“美国的执政团队强调的重点是要与国际合作、结盟强化供应链。”
 </p>
 <p class="p3">
 </p>
 <p>
 </p>
 <p class="p3">
 </p>
 <p class="p3">
  莫瑞：“我们每个人在这世界都互相依存，供应链也不例外。与全球伙伴合作，是我们在加强重供应链韧性方面的重中之重。今天有机会与台湾、日本、欧盟的伙伴，一同互动体现这种精神。”
 </p>
 <p class="p3">
  莫瑞指出，美国与盟友因为国家安全以及支持本国经济发展的义务，必须关注供应链相关问题。因为疫情已经大幅扰乱货品以及服务供应，导致价格上涨、原物料短缺，影响民生。今年
  <span class="s2">
   6
  </span>
  月
  <span class="s2">
   8
  </span>
  日白宫发表一份“跨部会关键供应链的评估报告”，其中就点出美国执政团队最重视的四大关键产业，半导体、大容量电池、药品、重要矿物。
 </p>
 <p class="p3">
  莫瑞：“这些产业各自有部分产能但是无法依靠自己制造所有的产品，在某些领域也过度依赖单一来源。事实上我们不可能靠一个国家一己之力，涵盖所有先进制造业的产能。”
 </p>
 <p class="p3">
  莫瑞指出，美国非常重视与台湾在供应链的关系，台湾是全球最大的半导体制造重镇，美国非常感谢台湾全力缓解全球芯片荒；同时，日本也是美国非常重要的盟友，去年供应链出现状况时，美国也与日本站在一起。此外，美国也持续与欧盟扩大在供应链的合作，美欧领导人要在半导体方面加强合作伙伴关系。
 </p>
 <p class="p3">
  <strong>
   郦英杰：台湾在全球供应链扮演举足轻重角色
  </strong>
 </p>
 <p class="p3">
  即将卸任的美国在台协会台北办事处长郦英杰在致词中强调，有一件事是无庸置疑的：台湾已经一再展现，在
  <span class="s3">
   迈
  </span>
  向更加永续的全球经济的路途上，台湾是可靠的伙伴，也是关键的要角。“不论是提供值得信赖的医疗设备，或是作为汽车生产线的半导体供应商，台湾都是不可或缺的合作伙伴。透过这些努力，台湾公司也会在增进并落实全球供应链韧性及打造未来新供应链方面，持续扮演举足轻重的角色。”
 </p>
 <p class="p3">
  郦英杰也提及，两天前，美国捐赠台湾的
  <span class="s2">
   250
  </span>
  万剂莫德纳疫苗已抵达台湾。美国人民永
  <span class="s3">
   远
  </span>
  不会忘记，当美国经历疫情最严峻的黑暗时刻，台湾慷慨驰援向美国捐赠了口罩及其他医疗物资。郦英杰说，“我们十分珍惜美台之间朋友有难时、互相帮助的长期传统。”
 </p>
 <p class="p3">
  除了从美国得到
  <span class="s2">
   250
  </span>
  万剂莫德纳疫苗外，地主国台湾经济部长王美花也感谢从
  <span class="s2">
   Covax
  </span>
  欧盟购买到新冠肺炎疫情疫苗、日本捐赠
  <span class="s2">
   124
  </span>
  万剂疫苗让台湾民众可以陆续开始施打疫苗。“各位盟友是我们真朋友、真进展的友谊。”
 </p>
 <p class="p3">
  <strong>
   台湾经长感谢美欧日真朋友
   <span class="s2">
   </span>
   捐疫苗送暖
  </strong>
 </p>
 <p class="p3">
  王美花进一步指出，台湾五月疫情爆发，在竹南包括京元电子
  <span class="s2">
   (
  </span>
  台湾半导体封装测试大厂
  <span class="s2">
   )
  </span>
  发生群聚感染的现象，台湾在进行快筛外，还迅速分舱、分流，在最短时间控制疫情，避免对全球半导体的供应链产生冲击。“从去年以来
  <span class="s2">
   Covid-19
  </span>
  也让各国政府用战略思维，来思考未来的经贸政策跟供应链的安全，加速全球供应链的改变。”
 </p>
 <p class="p3">
  王美花强调，台湾跟美、日、欧盟共享民主、人权、法治、自由经济的价值，产业间有非常紧密的连结，彼此可以持续、信赖的伙伴。“台湾高效率的代工半导体，已经是美国资讯业非常重要的后盾。日本则是台湾长久以来非常重要的技术来源、投资的来源国，在半导体、
  <span class="s2">
   5G
  </span>
  深化合作，尤其日本半导体的材料，是台湾最重要的来源，也相继在台湾投资、扩产。欧盟国家在台湾发展的离岸风电，已经成为台湾非常重要的伙伴。”王美花重申，“台湾是美、日、欧盟建立安全供应链非常值得信赖的伙伴。”
 </p>
 <p class="p3">
  欧盟经贸办事处长高哲夫指出，欧盟希望能增加在全球半导体产量的占比，特别是
  <span class="s2">
   5
  </span>
  奈米以下的先进半导体制程，台湾在这方面是引领的地位。在追求这些目标的同时，强调共同价值的重要性，与志同道合的伙伴合作的重要性。高哲夫也提到
  <span class="s2">
   Team Europe
  </span>
  ”
  <span class="s2">
   (
  </span>
  欧洲队
  <span class="s2">
   )
  </span>
  一起捐助许多物资帮助其他伙伴，让疫苗抵达需要的国家，这只是一个起步，我们必须持续合作才能取得成功，期待与全球伙伴打造更有韧性的供应链。
 </p>
 <p class="p3">
  日本台湾交流协会副代表星野光明认为，借着这次论坛显示必须进一步进行国际合作，让供应链更具备韧性，因应现在的挑战。全球的疫情需要全球通力合作，无法靠单一国家单打独斗。
 </p>
