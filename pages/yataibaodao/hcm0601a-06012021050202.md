 <p class="p3">
  在香港、澳门都禁止纪念六四相关活动之后，台湾公民团体发起六四当天在台北举行线上悼念晚会，并呼吁台湾责无旁贷延续六四纪念精神。台湾人权促进会秘书长施逸翔强调，六四学运并不是动乱，六四屠
  <span class="s3">
   杀
  </span>
  是中共建党百年不能回避的责任。
 </p>
 <p class="p3">
  包括华人民主书院、台湾人权促进会等多个台湾公民团体
  <span class="s2">
   1
  </span>
  日召开记者会，宣布在台举办六四悼念活动。
 </p>
 <p class="p3">
  华人民主书院倡议主任郭历轩指出，今年香港、澳门都禁止申请纪念活动，以致亚洲
  <span class="s2">
   (
  </span>
  华人
  <span class="s2">
   )
  </span>

 </p>
 <p class="p3">
  郭历轩：“今天在台湾面临
  <span class="s2">
   Covid-19
  </span>
  防疫压力，台湾作为合法集会的国家，我们依然会持续追究中国在六四的政治责任。同时声援中国制造的人权受害者以及更多良心犯。”
 </p>
 <p class="p3">
  <strong>
   六四是中国建党百年不能回避的责任
  </strong>
 </p>
 <p class="p3">
  台湾人权促进会秘书长施逸翔提及，六四受难者家属团体“天安门母亲”前一日不畏中共极权的压力，再次发表
  <span class="s2">
   32
  </span>
  周年祭文，祭文直指六四屠
  <span class="s3">
   杀
  </span>
  事件的核心，他们担心中国境内的年轻人不知道，甚至遗忘六四天安门事件，并重申六四学运并不是动乱，六四屠
  <span class="s3">
   杀
  </span>
  是中共建党百年不能回避的责任。
 </p>
 <p class="p3">
  施逸翔：“正因为中国境内还有像天安门母亲这样的良心，在中国境外各种抗中运动，尤其身处自由民主台湾，没有理由跟借口保持沉默，甚至逐渐遗忘
  <span class="s2">
   32
  </span>
  年前这场惊动全球的人权压迫事件。”
 </p>
 <p class="p3">
  施逸翔感叹，如果当年六四学运胜利，中国开始走向民主宪政体制，中国境内异议人士及社会运动开展中国的公民社会，也就不会有乔治欧威尔小说提到的严密的监控。维权律师可以实践民主法治的精神，流亡藏人能从流亡之地重回家乡，新疆维吾尔人也不会遭到再现纳粹集中营的再教育营对待，港人不会因为言论自由而被判处颠覆国家政权罪，也能选出自己的特首，自然不会发生反送中运动及形同实质戒严的《香港国安法》。中国就不会以极权扩张的方式，透过“一带一路”深层影响许多国家。
 </p>
 <p class="p3">
  施逸翔：“
  <span class="s2">
   2021
  </span>
  年的此刻，抗中已经不是两岸三地之事，抗中已成为国际社会鲜明的战线，作为在这抗中轴线下，台湾的公民社会及人权团体不仅大声站出来悼念六四事件，一起串连“奶茶联盟”对抗中国的坦克车，要与西藏人民、港人、新疆维吾尔人、缅甸人民，对抗与中共政权站在一起的集权政府。”
 </p>
 <p class="p3">
  <strong>
   停止纪念六四
   <span class="s2">
   </span>
   极权独裁输出不会停止
  </strong>
 </p>
 <p class="p3">
  民进党立委洪申翰指出，今年是六四事件
  <span class="s2">
   32
  </span>
  周年，到目前为止中共依旧没认错、没道歉，也没为受害者平反。中国共产党与过往相同又在六四前夕，抓捕了中国民运人士，只因为担心害怕在六四当天发出不一样的声音，今年黄晓敏、王爱忠又被拘捕甚至判罪，这都是令人难过伤心的事情。
 </p>
 <p class="p3">
  洪申翰：“这样的人权迫害不只发生在中国，香港已连续两年被禁止纪念六四，过去纪念六四在香港，不管是民主政治发展、人权运动是非常重要的传统，现在只要参与相关活动及举起标语，都会被用非法集会方式拘捕，很明显地这个极权正在扩张之中，如果我们无法彼此提醒甚至进一步反制它，这种极权独裁输出就不会停止，甚至会影响到每一个深爱的人。”
 </p>
 <p class="p3">
  西藏台湾人权连线理事长札西慈仁表示，流亡中的西藏人已经能选择自己的政府，但是在西藏里的西藏人、中国里的中国人完全没有机会。所以他要以西藏人身份呼吁，所有中国人出来不要忘了六四，同时我们藏人不可能忘记“
  <span class="s2">
   316
  </span>
  ”，团结的力量有一天会达到目标。
 </p>
 <p class="p3">
  <strong>
   六四洗白文出笼
   <span class="s2">
   </span>
   台湾民主实验室提反制
  </strong>
 </p>
 <p class="p3">
  台湾民主实验室研究员陈韵如提及他们观察到，每到六四就会会看到网上有大量洗白言论，像是坦克人没有被压死，证明解放军对人民的爱；或是中国成为世界工厂，全靠六四有镇压等。“台湾民主实验室除了专注资讯战的研究，也关注中国大外宣如何影响世界。去年靠近六四时，我们观察到许多关于六四洗白言论，我们警觉这些言论可能对民主造成伤害。归类起来有三类不正面回应、否认、合理化，我们推荐透过网站（
  <span class="s2">
   https://tiananmen-trolls.doublethinklab.org​
  </span>
  ）回应洗白言论。”
 </p>
 <p class="p3">
