 </p>
 <p>
  匈牙利政府今年四月与中国上海复旦大学签约，要提供首都布达佩斯南边第9区的大学城用地给复旦大学盖分校，还向中国贷款兴建校区，结果引发批评。布达佩斯市长卡拉松尼及第9区区长巴兰尼都反对此案，并把该地周边道路改名为达赖喇嘛道、光复香港道、维吾尔烈士径和谢仕光主教道，并于６月2日举行了揭牌仪式。
  <br/>
  <br/>
  “ 在布达佩斯有一条达赖喇嘛道路，我非常讶异。 ”披着西藏雪山狮子旗的匈牙利居民Karma Dorje在达赖喇嘛道的路牌前拍照，然后赶往新街名的揭牌仪式现场。
 </p>
 <p>
  <br/>
 </p>
 <p>
  <br/>
 </p>
 <p>
  匈牙利首都布达佩斯第9区市政府决定把计划兴建中国復旦大学分校地点周围的四条街道重新命名，分别是支持西藏流亡精神领袖的“ 达赖喇嘛道 ”（a Dalai Láma út）、抗议中国在香港实施国安法的“ 光复香港道 ”（Szabad Hongkong út）、纪念美国和多个西方国家认为中国在新疆实施种族灭绝的“ 维吾尔烈士径 ”（az Ujgur Mártírok útja）和声援被监禁的罗马天主教中国地下教会闽东教区主教“ 谢仕光主教道 ”（Hszie Si-kuang püspök út）。
  <br/>
  <br/>
  布达佩斯市长卡拉松尼（Gergely Karácsony）和第9区政府区长巴兰尼（Krisztina Baranyi）都出席了仪式，新闻发布会前，藏人团体还赠送了两人白色哈达和藏旗，卡拉松尼（Gergely Karácsony）称，如果复旦大学建成，他们将不得不忍受这些街道名称。
 </p>

  <br/>
  <br/>
  巴兰尼提出改路名的点子，她说，是为抗议匈牙利政府同意上海復旦大学在该区设立分校，并凸显北京侵犯人权行为。
  <br/>
  <br/>
  4月27日，匈牙利与中国复旦大学远端签署协议，将在首都布达佩斯市建立分校，预计2024年开学。
 </p>
 <p>
  反对声音针对规划问题，因为那块地原本是大学城，要新设大学生宿舍，而且对比2018年美国亿万富翁商人索罗斯（George Soros）创立的匈牙利最重要学术机构中欧大学因政治压力迁出，引来卡拉松尼强烈批评，他说，“政府现在要引进的代表中国共产党意识形态的一所大学，这所大学还要匈牙利纳税人缴纳数十亿欧元，这对布达佩斯和匈牙利来说是负面的。 ”
  <br/>
  <br/>
  自由主义反对派阵营的卡拉松尼表示，復旦大学的项目将使匈牙利在三十年前共产主义垮台时承诺的许多价值观受到怀疑，此外，复旦大学的章程要求代表中国共产党的世界观，属于非常严重的国家安全风险。
 </p>
 <p>

   </figcaption>

  <br/>
  <br/>
  这个欧盟国家设立的第一所中国大学分校，不只挑起了匈牙利国内政治争议。在欧盟层面，匈牙利已被视为中国和俄罗斯在欧盟和北约内部的“特洛伊木马”。在率先采购中国疫苗，又从中国贷款20亿美元建造一带一路计划的匈塞铁路等令欧盟不悦的事件之后，布达佩斯复旦分校计划是总理欧尔班(Orbán Viktor)“东方开放”外交政策的最新里程碑，再次引发欧盟关注。
  <br/>
  <br/>
  复旦分校校园建设资金总计约18亿欧元，比2019年匈牙利全国高等教育总支出13亿欧元还要多。当中的20%将从中央政府预算中拨划，剩下的资金来源是中国提供的15亿美元贷款，修建工程将包给中共国企中建集团。中建集团以不到10亿欧元的价格竞投该工程，并在投标中明确表示将使用一定数量的中国工人，并进口来自中国的建材。然而，中建集团过去的历史也引起了担忧。被美国列入从事贪腐与间谍活动黑名单的中建集团曾建造位于埃塞俄比亚的非洲联盟总部，该总部2017年1月被发现有伺服器的数据每晚都被传输到上海，此外还在桌子和墙壁中发现窃听设备和麦克风。
 </p>
