 <p>
  尽管一周之前，美国在阿拉斯加与中共政权的对话给了中共战狼们充分作秀的表演机会，让他们在认为美国政府好欺负的指导思想下，大过了一把嘴瘾。可是接下来的3月21号，美国、欧洲联盟，以及英国和加拿大等不约而同地宣布，对中共官员中的人权恶棍进行制裁等联手行动却清楚地表明：民主国家携手讨贼的大势已经形成，无论中共怎样使尽浑身解术也无法逆转。
 </p>
 <p>
  由于民主国家权力制衡的政治体系，做任何事必须多方协调，严谨而无法让效率太高。很明显，这次的制裁行动计划绝非开始于两个月前，而是去年蓬佩奥多次出访，协调达成共识行动的继续。不难看出，中共多年来不断破坏国际文明秩序，通过统战与外宣对自由世界渗透腐蚀的所作所为，已经在国际上引起了连锁反应。民主国家从要求外交对等，到联手反击，这只是必然反应的第一步而已。相信这种形势一定会步步为营地，迂回地向前推进。
  <br/>
  <br/>
  当然，看起来中共也毫不示弱，随即针对欧盟制裁行动采取了反制裁的报复。可是反观民主国家制裁的核心内容：冻结财产，禁止移民，拒发签证，金融系统跨国公司等不得与被制裁的人员个人及其所持有的公司和家人有业务往来等，在这些内容中，中共唯一能用的所谓“反制裁”手段，就是不给他们去中国签证。众所周知，被中共“反制裁”的欧美官员不可能有移民到中国的打算，也不会把财产存放在中国。因此，中方其它的内容都毫无用处。其实，对于这一点中共自己也清楚地知道，那为什么仍然要作出激烈的反应呢？一方面是欧美的制裁打到中共的痛点了，另一方面是要拉人民做挡箭牌和煽动、利用狭隘的民族主义情绪，既想借此转移国内矛盾，又想给自己被打肿的脸挽回一丝颜面。结果，中共这样的举动如同‘猴子学人行’，只是平添笑料罢了。
  <br/>
  <br/>
  随着越来越多中国人的觉醒，中共只能通过删帖、上门威胁等控制网络言论的手段，来让世人误认为中共在中国仍然有大量的支持者。在我们不能知道究竟有多少反对声音，网上所谓的“支持者”有多少是中共刻意安排的之前，仅仅从“真理部”允许留在网上的言论内容作出判断是不准确的。
  <br/>
  <br/>
  其实大国外交、对外政策的转向，一般人能够看到的只是经过长期酝酿之后的结果，而两国的真实较量早在酝酿之初就已经私下展开了。据我所知，之前中共为了阻止国际社会形成反共，甚至灭共联盟，几乎使出了浑身的解术，动用了所有能够动用的代理人施加影响，妄图使得民主联盟的形成胎死腹中。
  <br/>
  <br/>
  总之，天命不可违。过去几十年，中共完全有机会走上人路，把中国带入现代文明的行列。无奈，它一直沿著鬼途裸奔。如今人神共怒之下，灭共的大势已成，不可逆转。虽然不能指望“制裁”解决中国的专制问题，我们应该做也能够做的就是，与一切文明进步的国家和人们联手推进，抓住历史机遇，力求尽快地把中共独裁政权从人间铲除，送进地狱。只有这样，才能协助中国人民早日真正从专制的魔爪下解放出来，让中国人过上正常人的生活，也让世界根绝共产极权专制造成的一波又一波各种祸患。
 </p>
